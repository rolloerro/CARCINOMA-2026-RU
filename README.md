# 🧠 ClinicRecSkin Bot — CARCINOMA 2026 RU

Цифровой Telegram-помощник для врачей по клиническим рекомендациям при раке кожи (БКРК).

🚀 Быстрая навигация  
📘 Мгновенный доступ к PDF  
🎙 AI-ответы на текстовые и голосовые вопросы  
🩺 Практический инструмент для клинической работы 
🩻 Работает на всех устройствах 

👉 Запуск бота: https://t.me/ClinicRecS...

---

## 📌 Зачем создан бот

В клинической практике самое ценное — время.

ClinicRecSkin Bot помогает врачу:
- быстро находить нужный раздел рекомендаций
- получать номер страницы
- скачивать полный PDF-файл
- ориентироваться в диагностике и тактике лечения
- получать краткие AI-ответы на клинические вопросы

⚠️ В связи с возможным замедлением работы Telegram ответы могут приходить с задержкой до 10 секунд.

---

## 🔍 Возможности

### 📖 Навигация по разделам
Интерактивное содержание с постраничной разбивкой.

### 📥 Скачивание рекомендаций
Полный PDF-файл доступен прямо в чате.

### 🎙 AI-помощник
Поддержка текстовых и голосовых вопросов.
Ответы формируются на основе клинических рекомендаций.

### 📡 Контакты
Связь с разработчиком и поддержкой.

## 🚀 Установка

```bash
git clone https://github.com/rolloerro/CARCINOMA-2026-RU.git
cd CARCINOMA-2026-RU
npm install
Создать .env:

BOT_TOKEN=your_token_here
Запуск:

npm start
🔐 Безопасность
.env не публикуется

node_modules исключён из репозитория

PDF-файлы не хранятся в публичном доступе
